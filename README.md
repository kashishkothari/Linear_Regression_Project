This project demonstrates the Linear Regression to analyze the relationship between the different college students.
This dataset includes multiples features such as study hours, attendance, mid-term score, final score, previous gpa and gpa.

- Technologies Used

- Python
- Google Colab
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

- Model Evaluation
The model performance was evaluated using the following metrics:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score (Coefficient of Determination)

- Key Learning Outcomes
Through this project, I learned:

- Data preprocessing techniques
- Exploratory Data Analysis (EDA)
- Feature engineering and encoding
- Building and training a Linear Regression model
- Model evaluation using regression metrics
- Data visualization using Matplotlib and Seaborn
- Machine Learning workflow using Scikit-learn
- Implementing machine learning projects in Google Colab
